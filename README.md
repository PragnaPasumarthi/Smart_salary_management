# Smart Salary Management System

Smart Salary Management is a web-based application that helps users manage their monthly salary effectively by dividing income into six categories: Investment, Savings, Maintenance, Luxury, Education, and Emergency.

## Features
- Salary allocation using percentage sliders
- Visual charts for income distribution
- Expense tracking
- Savings and investment goals
- Wishlist for luxury purchases
- Financial reports with charts

## Technologies Used
- HTML
- CSS
- JavaScript
- Chart.js
